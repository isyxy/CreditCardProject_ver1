常見問題
1. P幣是什麼？
由「 Pi 拍錢包APP」所發放的 P幣，1 P幣 = 1元新臺幣，累積的 P幣可於「 Pi 拍錢包APP」消費付款時折抵使用。 Pi 拍錢包發送之
P幣的有效期限一般為180天，其他特殊活動所贈與的點數效期將視APP顯示為主。
Pi 拍錢包APP發送之 P幣僅限於 Pi 拍錢包APP付款折抵使用。
P幣並不為持卡人之資產，持卡人不得要求將累積之 P幣折合現金、兌換成紅利點數或其它使用方式。
詳盡 P幣回饋&使用規則，悉依「玉山 Pi 拍錢包信用卡回饋機制」為準。
2. 要如何獲得 P幣？
Pi 拍錢包信用卡持卡人須註冊 Pi 拍錢包App，並綁定於 Pi 拍錢包App，身分驗證資料須與本行留存之資料相符，綁卡成功並申請玉山信用卡帳單e化始享P幣回饋，正附卡人均可參加P幣回饋計劃。
Pi 拍錢包信用卡一般消費回饋 Pi 拍錢包App的P幣 (1 P幣＝1元)，將於消費請款後回饋至持卡人 Pi 拍錢包App的P幣帳戶。
3. P幣回饋如何計算？
Pi 拍錢包信用卡一般消費每筆消費採四捨五入法計算P幣回饋，正附卡分開計算，分期交易以總金額計算回饋1次。
2024/1/1起特店分期交易不享原卡片基本回饋1% P幣
(詳見公告)
4. P幣回饋如何查詢？
Pi 拍錢包APP贈送的 P幣將於生效後顯示於 Pi 拍錢包APP的「P幣帳戶」單元。操作步驟如下：
Step1. 打開 Pi 拍錢包APP點選下方選單「我的」
Step2. 點選藍色區塊的「P幣帳戶」
Step3. 即可切換「加點」與「扣點」查詢 P幣累兌紀錄與效期
5. P幣是否有使用期限？
Pi 拍錢包APP發送之 P幣於入帳後180天內須使用完畢，超過使用期限點數將自動歸零失效，不得要求兌換成現金或兌換成紅利點數其他特殊活動所贈與的點數效期將視APP顯示為主。
6. P幣要如何使用？
P幣於「Pi 拍錢包」APP內合作通路、停車繳費功能與點餐等功能均可進行消費抵用。透過 Pi 拍錢包P幣直接扣抵支付交易，並且可100%全額折抵消費金額至0元。當折抵消費時如P幣不足支付，剩餘部分將採用您選擇之付款方式進行交易。

9. 使用 P幣折抵時，有扣抵順序嗎？
P幣折抵時採先進先出法，將會從較早生效的點數進行扣除，依序折抵至點數歸零為止。
10. 要如何查詢我的 P幣點數？
使用者可透過 Pi 拍錢包APP的「P幣帳戶」，來查看您所剩餘的點數與剩餘效期。
11. 取消交易後，P幣點數會如何處理？
當持卡人取消交易時，付款時所抵⽤的 P幣，將依其退款金額計算應退還之 P幣並退還至 Pi 拍錢包APP會員帳號，退還的點數效期將自點數返還日起重新計算180天。
該筆交易原獲得之回饋點數，將依其退還款項金額計算應扣回之點數，並於 Pi 拍錢包APP會員帳號扣除 P幣點數。若帳號中的P幣點數不足以致無法扣回，玉山銀行得以1點換算為新臺幣1元後，列示為信用卡帳單之應繳金額向持卡人收取。
若進行部分退款，將先退還 P幣折抵的金額，再退還以信用卡支付的金額。
12. 換手機後，之前獲得的點數怎麼辦？
Pi 拍錢包APP P幣帳戶係生效於首次綁定的 Pi 拍錢包APP帳號中，如會員手機號碼改變了，請使用 Pi 拍錢包APP中「聯絡我們」功能與 Pi
拍錢包APP客服人員聯繫進行手機號碼變更作業，號碼變更程序完成後，即可以新手機號碼登入您的原 Pi 拍錢包APP會員帳號，並可完整保留至帳號中的 P幣。只要您不刪除 Pi 拍錢包APP會員帳號，點數就不會消失。

Pi 拍錢包APP P幣帳戶中會顯示近30天失效的 P幣數量，請盡快使用完畢，若已過期將全數失效且不得要求補發或兌現。
15. 信用卡新戶與舊戶的認定標準？
新戶定義為正卡核卡日前六個月內未持有玉山銀行任ㄧ張流通正卡卡片 (含認同卡、聯名卡) ，其餘則視為舊戶。
16. P幣如何折抵消費？
P幣可直接於 Pi 拍錢包APP內的交易流程直接操作扣抵，並且最高可100%折抵消費金額至0元。
折抵消費時若 P幣不足支付，剩餘部分將採用您選擇之付款方式進行交易。
17. 使用 Pi 拍錢包APP的個人交易是否可以獲得 P幣？
Pi 拍錢包APP的個人交易付款功能，恕不提供贈送 P幣點數與使用 P幣折抵付款金額之功能。
18. 註銷 Pi 拍錢包APP的帳號時，剩餘點數怎麼辦？
註銷前需先將點數使用完畢。當重新註冊新帳號，綁同一張卡時，點數將重新計算。

10. 顧客參加玉山Unicard的「任意選」方案，將瓦城美食納入其中，並在百貨公司館內的瓦城泰國料理餐廳消費，但事後帳單卻無瓦城的加碼回饋？
餐飲類別的消費如果在百貨公司、飯店、酒店、旅館、賣場、商場、車站及美食街內設立之特店，以致信用卡簽單及帳單名稱無法判別為該餐廳，將無法列入回饋計算，舉例：新光三越館內的瓦城泰國料理，其刷卡簽單及信用卡帳單上顯示之商店名稱為新光三越，非瓦城泰國料理，即無法符合瓦城美食消費。
11. 國外實體消費的定義是什麼？
國外實體消費係指註冊地在國外之實體特店，被收取「國外交易服務費」之實體商店交易，並使用實體信用卡或感應式 (例如：Apple Pay、Samsung Pay、Google Pay等消費)之面對面交易；不包含國外購物網站、國外訂房網店等線上交易。
12. 一般消費項目不含歐洲地區實體特店交易，玉山Unicard百大指定消費其中的國外實體消費是否有回饋呢？

13. 什麼時候可以拿到玉山e point回饋？結帳日是否會影響玉山e point回饋時間呢？
玉山e point回饋計算時間與帳單結帳週期/結帳日並無相關。
玉山e point回饋包含「一般消費回饋」與「百大指定消費」加碼回饋，兩者的回饋時間不同：
一般消費回饋：
※適用消費：一般消費資格說明請見Q3。
※回饋時間：店家請款後即時回饋予正卡持卡人(正附卡合併計算)。
百大指定消費加碼回饋：
※適用消費：具一般消費資格，並符合百大指定消費，詳細請見Q3和Q4。
※消費時間：依每月(日曆月)計算，且消費需於次月10日前請款。
※回饋時間：根據消費當月最終選取的方案計算後，次月17日發放回饋。
※舉例說明：8/1~8/31消費且店家請款日為8/1~9/10之交易，將於9/17回饋。若店家超過9/10請款，則會於次次月17日回饋予正卡持卡人，即為：消費日8/31，請款日為9/12，則將於10/17回饋。
你可能想進一步詢問：
月底刷卡，但次月才請款的交易，怎麼計算回饋？
消費店家只要在次月10日以前請款，都會根據當月的方案給予回饋，回饋上限亦為根據當月方案而訂。
月中更換方案，會影響玉山e point回饋計算？
不影響，玉山e point回饋依據該月最後一日最終切換的方案給予回饋。
為什麼我在百大指定消費清單中的店家消費，卻只收到1%玉山e point回饋？
一般消費的1%回饋與百大指定消費加碼回饋的發放時間不同。百大指定消費加碼回饋會在次月17日發放回饋。

常見問題
常見問題
什麼是日圓雙幣卡?
提供顧客使用「兩種幣別-臺幣與日幣」付款的信用卡卡別。
在國內消費(台灣)，使用臺幣帳戶扣款；在國外消費(非台灣)使用外幣帳戶中日幣幣別扣款。
日圓雙幣卡在日本消費有什麼優勢?
在日本進行消費時本行會直接從約定外幣帳戶扣款日幣，若遇到外幣存款餘額不足導致自動扣繳失敗，將由約定之臺幣帳戶依扣繳失敗當日匯率轉換為日幣，存入約定之外幣帳戶扣繳日幣應付帳款。
*省錢小秘訣：可以趁日幣匯率低點進行換匯存入外幣帳戶，利用匯差存旅遊金。
申請玉山熊本熊日圓雙幣卡的條件為?
須開立本行臺、外幣帳戶及申請外幣帳戶自動轉帳扣繳。若還未擁有本行外幣帳戶，請先辦理
線上開戶
待開戶完成後即可
申辦
熊本熊日圓雙幣卡連結外幣帳戶自動扣繳。
玉山熊本熊一般信用卡和日圓雙幣卡的差別?
「一般信用卡」毋須開立本行臺外幣帳戶及申請外幣帳戶自動轉帳扣繳。
「日圓雙幣卡」須開立本行臺外幣帳戶及申請外幣帳戶自動轉帳扣繳。
在日本刷卡消費時，店家要求我輸入PIN碼怎麼辦?
在日本信用卡交易原則上是不需要輸入密碼的，惟因少數海外收單行需輸入密碼驗證，可以先嘗試請商店跳過此步驟，如無法跳過請輸入【預借現金密碼】或【生日月日四碼】作為交易識別密碼，惟檢核與否將依當下交易情境必要性動態調整，一般消費輸入預借現金密碼仍視同一般交易。
在韓國刷玉山熊本熊日圓雙幣卡如何計算匯率?
若使用玉山熊本熊日圓雙幣卡於韓國消費，將依國際組織JCB依約所列示之結匯日大盤匯率，先以韓元(KRW)轉換為國際貨幣美元(USD)，再轉換為指定約定日圓(JPY)結付。
首刷禮行李箱寄送時間為何?
符合首刷禮資格當期關帳後約3個月內將贈品寄送至正卡持卡人之信用卡帳單地址。
若帳單地址需異動，煩請於首刷禮累積消費金額達指定消費門檻前致電玉山銀行客服(02)21821313 或
玉山銀行智能客服
進行修改，恕無法指定贈品寄送日期、時間。
