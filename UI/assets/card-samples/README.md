# 信用卡樣本圖片資料夾

此資料夾用於存放信用卡的樣本圖片，供圖片識別功能參考或訓練使用。

## 使用方式

### 1. 圖片識別功能

目前的 CameraScanner 組件使用遠程 API 進行卡片識別：
- API 端點：`http://192.168.1.10:8000/scan`
- 方法：POST
- 格式：JSON (`{ "image": "base64_encoded_image" }`)

### 2. 新增樣本圖片

如果您想新增信用卡樣本圖片：

1. **拍攝或收集信用卡圖片**
   - 確保光線充足
   - 卡片完整且清晰可見
   - 建議使用 JPG 或 PNG 格式

2. **命名規則**
   ```
   {銀行名稱}_{卡片名稱}_{編號}.jpg

   例如：
   - 中國信託_OPENPOINT聯名卡_01.jpg
   - 台新銀行_Richart卡_01.jpg
   - 玉山銀行_Unicard_01.jpg
   ```

3. **將圖片放入此資料夾**
   - 每張卡片建議準備 5-10 張不同角度、光線的照片
   - 用於訓練或參考

## 目前支援的信用卡

系統目前支援以下信用卡：

1. 中國信託 OPENPOINT 聯名卡
2. 中國信託 LINE Pay 卡
3. 台新銀行 Richart卡
4. 玉山銀行 Unicard
5. 國泰世華 CUBE卡

## 訓練模型（可選）

如果需要本地識別功能而非使用遠程 API：

1. 收集足夠的樣本圖片（每張卡至少 50-100 張）
2. 使用 YOLO 或其他物件偵測模型進行訓練
3. 部署模型到本地或雲端服務器
4. 更新 `CameraScanner.tsx` 中的 API_URL

## 注意事項

⚠️ **隱私保護**
- 請勿上傳包含個人資訊的信用卡圖片（卡號、姓名、CVV等）
- 僅使用信用卡正面的設計樣本
- 建議使用銀行官方提供的卡面圖片或展示樣本

⚠️ **版權聲明**
- 信用卡圖片版權歸各銀行所有
- 僅供個人學習和開發使用
